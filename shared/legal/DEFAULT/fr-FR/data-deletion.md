## Vos donnees, votre choix

Vous avez le droit de demander la suppression des donnees personnelles associees a votre compte sur **winegod.ai**.

## Ce qui est supprime

Lorsque vous supprimez votre compte, nous supprimons de maniere definitive :

- Vos donnees de profil (nom, e-mail, photo de profil)
- Le lien avec votre fournisseur de connexion (Google, Facebook, Apple ou Microsoft)
- L'integralite de votre historique de chat avec Baco
- Toutes les conversations enregistrees (favoris)

## Ce qui n'est pas supprime

- Les journaux d'usage anonymises (sans lien avec votre compte), conserves pour l'analytique agregée du produit
- Les donnees stockees localement dans votre navigateur - elles sont effacees automatiquement lors de la suppression du compte, mais vous pouvez aussi les supprimer manuellement dans les reglages du navigateur

## Donnees detenues par des tiers

Supprimer votre compte sur winegod.ai supprime vos donnees de nos systemes. Pour revoquer l'acces de winegod.ai au sein de votre fournisseur de connexion, utilisez les reglages de securite du fournisseur :

- **Google:** Parametres du compte > Securite > Applications tierces
- **Facebook:** Parametres > Applications et sites web
- **Apple:** Parametres > Apple ID > Apps utilisant Apple ID
- **Microsoft:** Compte > Confidentialite > Apps et services

## Questions

Pour toute question sur la suppression des donnees ou la confidentialite, contactez-nous a [privacy@winegod.ai](mailto:privacy@winegod.ai).
